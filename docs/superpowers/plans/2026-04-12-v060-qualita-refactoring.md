# v0.6.0 "Qualita' e Refactoring" Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Refactor the TrekTrak codebase for maintainability — split InteractiveMap.tsx, eliminate prop drilling with Zustand UI store, add React component smoke tests, and migrate PWA from deprecated next-pwa to @serwist/next.

**Architecture:** Four independent refactoring streams executed in order: (1) split the 704-line InteractiveMap into focused modules, (2) create a Zustand uiStore for tool state to replace prop drilling, (3) add smoke tests for 6 key components, (4) swap PWA library. Each stream must leave the app functionally identical.

**Tech Stack:** Next.js 15, TypeScript, Zustand, React Testing Library, Jest, @serwist/next, Leaflet/react-leaflet, Recharts.

**Spec:** `docs/superpowers/specs/2026-04-12-v060-qualita-refactoring-design.md`

### Task 4: Create `src/stores/uiStore.ts`

**Files:**
- Create: `src/stores/uiStore.ts`

- [ ] **Step 1: Create the store**

```ts
// src/stores/uiStore.ts
import { create } from 'zustand';

interface UIState {
  compassActive: boolean;
  rulerActive: boolean;
  quizActive: boolean;
  progressOpen: boolean;
  drawerOpen: boolean;
  searchOpen: boolean;

  toggleCompass: () => void;
  toggleRuler: () => void;
  toggleQuiz: () => void;
  deactivateCompass: () => void;
  deactivateRuler: () => void;
  openProgress: () => void;
  closeProgress: () => void;
  setDrawerOpen: (open: boolean) => void;
  setSearchOpen: (open: boolean) => void;
}

export const useUIStore = create<UIState>((set) => ({
  compassActive: false,
  rulerActive: false,
  quizActive: false,
  progressOpen: false,
  drawerOpen: false,
  searchOpen: false,

  toggleCompass: () => set((s) => ({
    compassActive: !s.compassActive,
    rulerActive: false,
    quizActive: false,
  })),
  toggleRuler: () => set((s) => ({
    rulerActive: !s.rulerActive,
    compassActive: false,
    quizActive: false,
  })),
  toggleQuiz: () => set((s) => ({
    quizActive: !s.quizActive,
    compassActive: false,
    rulerActive: false,
  })),
  deactivateCompass: () => set({ compassActive: false }),
  deactivateRuler: () => set({ rulerActive: false }),
  openProgress: () => set({ progressOpen: true, quizActive: false }),
  closeProgress: () => set({ progressOpen: false }),
  setDrawerOpen: (open) => set({ drawerOpen: open }),
  setSearchOpen: (open) => set({ searchOpen: open }),
}));
```

  });
